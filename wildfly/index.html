Подключение WildFly

  1. Скачиваем сервер WildFly (Download)
  2. Создаем Maven проект
  3. Проверяем наличия файла src/main/webapp/WEB-INF/web.xml
  4. Заполняем файл web.xml
    <?xml version="1.0" encoding="UTF-8"?> <web-app version="2.4" xmlns="http://java.sun.com/xml/ns/j2ee" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://java.sun.com/xml/ns/j2ee http://java.sun.com/xml/ns/j2ee/web-app_2_4.xsd"> </web-app>
  5. Нажимаем Add/Edit Configuration (находится справа от Build Project)
  6. Нажимаем Add New Configuration (+)
  7. Находим JBoss Server
  8. Выбираем Local
  9. Нажимаем на кнопку Cofigure
  10. Нажимаем на Add Application Server (+)
  11. Указываем путь до папки - JBoss Home
  12. В Run/Debug Configuration выбираем JRE
  13. Убираем Build в Before launch - нажатием на кнопку Remove (-)
  14. Затем с помощьюю Maven собираем проект clean, package, чтобы указать Idea какой архив/артефакт необходимо запускать
  15. Нажимаем Add/Edit Configuration (находится справа от Build Project)
  16. Переходим на вкладку Deployment
  17. Нажимаем на кнопку Add (+)
  18. Выбираем External Source
  19. Указываем путь до war архива/артефакта, который был скомпилирован ранее Maven
  20. Запускаем через Debug
